qdrant plugin for `Tutor <https://docs.tutor.edly.io>`__
########################################################

a plugin to run qdrant vector db


Installation
************

.. code-block:: bash

    pip install git+https://github.com/Mista-Kev/openedx-tutor-rag.git

Usage
*****

.. code-block:: bash

    tutor plugins enable qdrant
